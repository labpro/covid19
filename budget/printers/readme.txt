# Introdução
------------

Este é projeto temporário para compilar um documento orçamentário afim
de  conseguir  recursos para que o LabPRO dê assistência no combate ao
Corona Virus. 

Os arquivos em txt são inicialmente por comodidade para que depois sejam
passados para um pdf com formatação adequada.
